# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
- 停车场管理
- 车位管理
- 车辆出入缴费
- 协同过滤算法推荐车位信息

## 适用场景
- 停车场管理
- 车位分配
- 车辆出入管理
- 缴费系统

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
